============================ CodeWar : Compilation ===========================

Pour compiler ce projet vous aurez besoin de l'utilitaire cmake qui permet de générer des makefile.

# Prerequis:
- cmake
- libgtk-3-dev

# Build from sources

## Compilation: 

```
mkdir build
cd build
cmake ..
make
```

ou

```
./build.sh
```

## Execute
```
    build/src/main -> GUI Mode
    build/src/terminal -> Terminal Mode
```
